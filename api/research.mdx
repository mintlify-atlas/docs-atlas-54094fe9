## Overview

The Research API allows you to submit research tasks that run asynchronously in the background. You can research a topic, analyze specific sources, or both. Each task returns a task ID that you can use to track progress and retrieve results.

## Submit Research Task

<Card title="POST /research" icon="flask">
  Submit a new research task for topic or sources analysis
</Card>

### Request

<ParamField body="notebook_id" type="string" required>
  The notebook ID to store the research results
</ParamField>

<ParamField body="topic" type="string">
  The research topic to investigate (minimum 3 characters). Either `topic` or `sources` must be provided.
</ParamField>

<ParamField body="sources" type="array">
  Array of sources to research. Either `topic` or `sources` must be provided.
  
  <Expandable title="ResearchSource object">
    <ParamField body="source_type" type="enum" required>
      Type of source. One of: `URL`, `MANUAL`, `UPLOAD`
    </ParamField>
    
    <ParamField body="source_url" type="string">
      The URL of the source (required for `URL` type)
    </ParamField>
    
    <ParamField body="source_content" type="string">
      The content of the source (for `MANUAL` or `UPLOAD` types)
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="task_id" type="string">
  Unique identifier for the submitted task
</ResponseField>

<ResponseField name="notebook_id" type="string">
  The notebook ID where results will be stored
</ResponseField>

<ResponseField name="status" type="string">
  Current task status: `IN_QUEUE`
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable status message
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X POST https://api.decipherit.com/research \
  -H "Content-Type: application/json" \
  -d '{
    "notebook_id": "nb_123456",
    "topic": "quantum computing applications"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.decipherit.com/research",
    json={
        "notebook_id": "nb_123456",
        "topic": "quantum computing applications"
    }
)

data = response.json()
print(f"Task ID: {data['task_id']}")
```

```javascript JavaScript
const response = await fetch('https://api.decipherit.com/research', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    notebook_id: 'nb_123456',
    topic: 'quantum computing applications'
  })
});

const data = await response.json();
console.log(`Task ID: ${data.task_id}`);
```
</CodeGroup>

### Example Response

```json
{
  "task_id": "task_abc123",
  "notebook_id": "nb_123456",
  "status": "IN_QUEUE",
  "message": "Research task submitted and will be processed."
}
```

### Error Responses

<ResponseField name="400 Bad Request" type="object">
  <Expandable title="Error details">
    Returned when:
    - `notebook_id` is missing or empty
    - Both `topic` and `sources` are missing or empty
    
    ```json
    {
      "detail": "Notebook ID is required."
    }
    ```
    
    ```json
    {
      "detail": "Either topic or sources must be provided."
    }
    ```
  </Expandable>
</ResponseField>

## Get Task Status

<Card title="GET /research/{task_id}" icon="clock">
  Retrieve status and results of a research task
</Card>

### Request

<ParamField path="task_id" type="string" required>
  The unique task identifier returned when submitting the research task
</ParamField>

### Response

<ResponseField name="task_id" type="string">
  Unique identifier for the task
</ResponseField>

<ResponseField name="notebook_id" type="string">
  The notebook ID where results are stored
</ResponseField>

<ResponseField name="topic" type="string">
  The research topic (if provided)
</ResponseField>

<ResponseField name="sources" type="array">
  Array of research sources (if provided)
</ResponseField>

<ResponseField name="status" type="string">
  Current task status: `IN_QUEUE`, `IN_PROGRESS`, `COMPLETED`, or `FAILED`
</ResponseField>

<ResponseField name="created_at" type="datetime">
  ISO 8601 timestamp when the task was created
</ResponseField>

<ResponseField name="result" type="object">
  Task results (available when status is `COMPLETED`)
</ResponseField>

<ResponseField name="error" type="string">
  Error message (available when status is `FAILED`)
</ResponseField>

<ResponseField name="completed_at" type="datetime">
  ISO 8601 timestamp when the task completed (if applicable)
</ResponseField>

<ResponseField name="failed_at" type="datetime">
  ISO 8601 timestamp when the task failed (if applicable)
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X GET https://api.decipherit.com/research/task_abc123
```

```python Python
import requests

response = requests.get(
    "https://api.decipherit.com/research/task_abc123"
)

data = response.json()
print(f"Status: {data['status']}")
```

```javascript JavaScript
const response = await fetch('https://api.decipherit.com/research/task_abc123');
const data = await response.json();
console.log(`Status: ${data.status}`);
```
</CodeGroup>

### Example Response (In Progress)

```json
{
  "task_id": "task_abc123",
  "notebook_id": "nb_123456",
  "topic": "quantum computing applications",
  "sources": null,
  "status": "IN_PROGRESS",
  "created_at": "2026-03-03T10:30:00Z",
  "result": null,
  "error": null,
  "completed_at": null,
  "failed_at": null
}
```

### Example Response (Completed)

```json
{
  "task_id": "task_abc123",
  "notebook_id": "nb_123456",
  "topic": "quantum computing applications",
  "sources": null,
  "status": "COMPLETED",
  "created_at": "2026-03-03T10:30:00Z",
  "result": {
    "content": "Research findings...",
    "sources": [...]
  },
  "error": null,
  "completed_at": "2026-03-03T10:32:15Z",
  "failed_at": null
}
```

### Error Responses

<ResponseField name="404 Not Found" type="object">
  <Expandable title="Error details">
    Returned when the task ID does not exist
    
    ```json
    {
      "detail": "Task not found."
    }
    ```
  </Expandable>
</ResponseField>

## Submit Research with Sources

### Example: Research URLs

<CodeGroup>
```bash cURL
curl -X POST https://api.decipherit.com/research \
  -H "Content-Type: application/json" \
  -d '{
    "notebook_id": "nb_123456",
    "sources": [
      {
        "source_type": "URL",
        "source_url": "https://example.com/article1"
      },
      {
        "source_type": "URL",
        "source_url": "https://example.com/article2"
      }
    ]
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.decipherit.com/research",
    json={
        "notebook_id": "nb_123456",
        "sources": [
            {
                "source_type": "URL",
                "source_url": "https://example.com/article1"
            },
            {
                "source_type": "URL",
                "source_url": "https://example.com/article2"
            }
        ]
    }
)

data = response.json()
print(f"Task ID: {data['task_id']}")
```
</CodeGroup>
